# Bank Sampah Digital

## Deskripsi

Bank Sampah Digital adalah aplikasi berbasis web yang digunakan untuk mengelola data nasabah, transaksi setor sampah, penarikan saldo, serta pembuatan laporan secara digital.

## Fitur

- Login Admin
- Dashboard
- Data Nasabah
- Data Jenis Sampah
- Transaksi Setor Sampah
- Penarikan Saldo
- Laporan
- Logout

## Teknologi

- PHP
- MySQL
- Bootstrap
- XAMPP

## Instalasi

1. Clone repository

```bash
git clone https://github.com/username/bank-sampah-digital.git
```

2. Pindahkan project ke folder `htdocs`.

3. Jalankan Apache dan MySQL.

4. Buat database `bank_sampah`.

5. Import file `database/bank_sampah.sql`.

6. Jalankan aplikasi melalui browser.
